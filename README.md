# ATM Machine Simulator

A lightweight, terminal-based ATM Machine Simulator built using Python. This project demonstrates core programming principles including conditional routing, input handling, and dynamic arithmetic state operations.

## 🚀 Features

- **Check Balance:** View the current available funds safely.
- **Deposit Money:** Add money to the account, which instantly updates the global balance.
- **Withdraw Money:** Extract funds with real-time balance validation to prevent overdrafts.
- **Error & Input Handling:** Built-in safeguards against case-sensitivity and unrecognized commands.

## 🛠️ Tech Stack

- **Language:** Python 3.x

## 📋 Core Logic Demonstrated

- **Conditional Logic (`if-elif-else`):** Used to capture and evaluate user selection seamlessly.
- **Variable Mutation:** Correctly updates state variables to track mathematical changes without runtime scope conflicts.
- **Input Sanitization:** Uses Python string methods to normalize terminal entries.
